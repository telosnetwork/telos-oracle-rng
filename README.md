# Telos RNG Oracle

This repository contains the Antelope contract for the Telos Random Number Generation (RNG) Oracle.

## Requirements

NodeJS 14+ and NPM

## Setup

`npm install`

## Build

`bash build.sh`

## Deploy

`bash deploy.sh { mainnet | testnet | local }`
